# interacoes-gravitacionais-godot
Repositório dedicado à disponibilização do projeto desenvolvido para meu TCC.

## Controles:
- Segure o botão do meio do mouse para controlar a câmera;
- Clique nos planetas, ou na lista à direita da tela, para centralizar a câmera em outro planeta;
- Pressione ESC ou clique em "Retornar ao menu principal", à direita da tela, para retornar ao menu principal.

## Executar simulador:
Navegue até a aba "Releases", à direita do navegador de arquivos da página principal desse repositório, ou clique [aqui](https://github.com/Fefonfon/interacoes-gravitacionais-godot/releases) e baixe o arquivo executável correto para seu sistema operacional.

## Executar projeto:
Baixe a versão 4.6.X do motor gráfico Godot em: [https://godotengine.org/pt-br/](https://godotengine.org/pt-br/)
e realize a importação do projeto.
